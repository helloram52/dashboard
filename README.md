d3-workout
==========
```
This is a d3 workout repository using node.js at the backend. To try out, use the below.

# Get the latest repo
git clone https://bitbucket.org/vadivel_selvaraj/d3workout
cd d3workout

# Install NPM dependencies
npm install

# Start the server
npm start
